# How To Build

Video: https://asciinema.org/a/pLuJlQYPDFN1kOJMgfLPs66Cz

## Prerequisite
- GCC / G++
- CMake

## Steps

Create a build directory

```
mkdir build
```

Change path to build directory

```
cd build
```

Run `CMake` for create makefile

```
cmake ../src/
```

Make C++ Files

```
make
```

Run Application
```
./moodleodev
```
